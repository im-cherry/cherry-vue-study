# Simple Vue

## Vue.js 란?

- Vue 는 사용자 인터페이스를 만들기 위한 진보적인 프레임 워크 입니다.
- Vue 는 Component 기반의 SPA(Single Page Application) 사이트를 구축할 수 있습니다.
